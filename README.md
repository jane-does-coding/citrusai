# Auth Template

## Usage

Clone the project

```
git clone https://github.com/YevheniiaSimaka/Nextjs-Auth-Template
```

Install dependencies

```
npm i
```

Create .env

```
DATABASE_URL=
NEXTAUTH_SECRET=
```

Set up prisma

```
npx prisma generate
npx prisma db push
```

Run it

```
npm run dev
```
